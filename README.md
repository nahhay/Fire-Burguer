Fire Burguer

Site institucional desenvolvido para uma hamburgueria fictícia, com foco em apresentar o cardápio, informações da empresa e facilitar o contato com os clientes.
Acesse o projeto:

https://nahhay.github.io/Fire-Burguer/

Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* GitHub Pages

Funcionalidades

* Layout responsivo
* Cardápio organizado por categorias
* Navegação intuitiva
* Design moderno
* Botão de contato via WhatsApp
* Animações e efeitos visuais

Estrutura do projeto

```text
Fire-Burguer/
│
├── img/
├── index.html
├── style.css
├── script.js
└── README.md
```

Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/nahhay/Fire-Burguer.git
```

2. Acesse a pasta do projeto.

3. Abra o arquivo `index.html` em qualquer navegador.

Objetivo

Este projeto foi desenvolvido para praticar desenvolvimento front-end utilizando HTML, CSS e JavaScript, simulando um site real para uma hamburgueria.
